Требования к боту.

Бот для Telegram, который позволяет подключить свой Notion, указать страницу для записи Inbox, и дописывает туда любые сообщения, которые ему присылают.

Схема работы:

1. Авторизация
    1. Пользователь нажимает /start и бот спрашивает его о том, какой аккаунт подключить(возможно через авторизацию в Notion)
    2. Вторым шагом бот запрашивает страницу для записи - пользователь должен указать ссылку на страницу (либо название страницы)
2. Добавление заметки
    1. Пользователь пишет сообщение в чат с ботом, бот берет текст сообщения отправляет его к себе на бекэнд и по Notion API дописывает на указанную страницу
    2. В ответ в случае успеха бот отвечает “Заметка записана”, в случае ошибки - присылает сообщение с ошибкой.

Нефункциональные требования:

1. Бот должен работать в телеграм
2. Язык для бота любой - предпочтительно Go/Python
3. Бот должен уметь общаться с Notion API
4. Бот умеет обрабатывать ошики и присылать прямо в чат и нформацию об ошибке и что можно сделать.
